# Google BigQuery data source for Grafana

The Google BigQuery data source plugin allows you to query and visualize Google BigQuery data from within Grafana.

## Documentation

For complete documentation, refer to [Google BigQuery data source documentation](https://grafana.com/docs/plugins/grafana-bigquery-datasource/latest/).

- [Configure the data source](https://grafana.com/docs/plugins/grafana-bigquery-datasource/latest/configure/) — Authentication, provisioning, and Terraform setup
- [Query editor](https://grafana.com/docs/plugins/grafana-bigquery-datasource/latest/query-editor/) — SQL and visual query editors, macros, partitioned tables
- [Template variables](https://grafana.com/docs/plugins/grafana-bigquery-datasource/latest/template-variables/) — Dynamic dashboards with BigQuery queries
- [Annotations](https://grafana.com/docs/plugins/grafana-bigquery-datasource/latest/annotations/) — Overlay events on your graphs
- [Alerting](https://grafana.com/docs/plugins/grafana-bigquery-datasource/latest/alerting/) — Create alert rules using BigQuery data
- [Troubleshooting](https://grafana.com/docs/plugins/grafana-bigquery-datasource/latest/troubleshooting/) — Solve common issues

## Installation

1. Navigate to the [BigQuery plugin page](https://grafana.com/grafana/plugins/grafana-bigquery-datasource/) on Grafana.
2. Click **Install plugin**.

For detailed installation and configuration instructions, refer to [Configure the data source](https://grafana.com/docs/plugins/grafana-bigquery-datasource/latest/configure/).

## Requirements

The following Google APIs must be enabled in your GCP project:

- [BigQuery API](https://console.cloud.google.com/apis/library/bigquery.googleapis.com)
- [Cloud Resource Manager API](https://console.cloud.google.com/apis/library/cloudresourcemanager.googleapis.com)
